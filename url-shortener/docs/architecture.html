⚡ Cache-Aside Pattern (Redirect Flow)
GET /{shortCode}
    │
    ├──▶ Redis.get("url:{code}")
    │         │
    │      HIT ────▶ Return URL instantly < 1ms
    │         │
    │      MISS ───▶ MySQL.findByShortCode()
    │                    │
    │                    ├──▶ Validate (active? expired?)
    │                    ├──▶ Redis.set(key, url, TTL=24h)
    │                    └──▶ Return URL + Log access event
    │
    FAULT TOLERANCE: Redis down? → silently fall through to DB
🔌 API Endpoints
Method Path Description Status Codes
POST /api/shorten Create short URL. Supports custom alias + expiry.
201 Created 400 Bad Request 409 Conflict 429 Rate Limit
GET /{shortCode} Redirect to original URL. Tracks click + logs access event.
302 Found 404 Not Found 410 Gone (expired)
GET /api/stats/{code} Analytics: click count, trend chart, geo breakdown, recent clicks.
200 OK 404 Not Found
DELETE /api/{shortCode} Soft-delete. Preserves analytics. Evicts from Redis cache.
204 No Content 404 Not Found
